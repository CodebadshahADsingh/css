WHEN WE SET TH WIDTH AND HIEGHT PROPERTIES OF AN ELEMENT WITH CSS, WE NJUST SET THE WIDTH AND HIEGHT OF THE CONTENT AREA. TO CALCULATE THE FULL SIZE OF AN ELEMENT, WE MUST ALSO ADD PADDING BORDERS AND MARGINS



TOTAL ELEMENT WIDTH = WIDTH +LEFTPADDING+ RIGHT PADDING+ LEFT BORDER + RIGHT BORDER + LEFT MARGIN + RIGHT MARGIN



TOTAL ELEMENT HIEGHT= HEIGHT+TOP PADDING + BOTTOM PADDING+ TOP ORDER+ BOTTOM ORDER+ TOP MARGIN + BOTTOM MARGIN
